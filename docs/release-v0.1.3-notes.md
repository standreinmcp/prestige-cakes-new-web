# v0.1.3 — Produse la comandă

Catalog page for made-to-order products, sharing the vitrină layout via a reusable `CatalogPage` component.

## What's included

- `/produse-la-comanda` moved into the `(catalog)` route group (hero nav + banner)
- Shared `CatalogPage` component (refactored from `VitrinaCatalog`)
- `madeToOrderProducts` dataset with Figma copy and category filters
- Vitrină live updated to use `CatalogPage`
