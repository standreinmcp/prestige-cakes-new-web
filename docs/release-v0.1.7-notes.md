# v0.1.7 — Ridicare personală

Checkout variant with pickup selected by default — location info and map instead of delivery address fields.

## What's included

- `/checkout/ridicare` route with `defaultFulfillment="pickup"`
- Pickup location copy and embedded map widget
- Cash payment label: “Numerar la ridicarea comenzii”
- Order summary without delivery fee

`/checkout/ridicare`
